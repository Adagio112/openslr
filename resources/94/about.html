This paper introduces Multilingual LibriSpeech (MLS) dataset, a large multilingual corpus suitable for speech research. The dataset is derived from read audiobooks from LibriVox and consists of 8 languages - English, German, Dutch, Spanish, French, Italian, Portuguese, Polish. For each language, we provide two separate .tar.gz files: 1. mls_[language].tar.gz : Train, dev and test sets. Also, includes small training set for limited supervision (10hr, 1hr or 10 minutes of labelled speech). 2. mls_lm_[language].tar.gz : Language modelling corpus. More details on these files and their directory structure can be found in README files included in the .tar.gz files. You can cite the data using the following BibTeX entry:

NOTE: We have made few updates to the MLS dataset after our INTERSPEECH paper was submitted to include more #hours and also improve the quality of transcripts.
