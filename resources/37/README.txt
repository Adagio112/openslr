This data set contains multi-speaker high quality transcribed audio data for
Bengali. The data set consists of wave files, and a TSV file. There are two
zip files, one for each local which contain a file: line_index.tsv and the wave
files. Line index has a fileID and the transcription.

The data set has been manually quality checked, but there might still be errors.

This data set was collected by Google.
